Project Overview


Passionate about elevating user experiences, I revamped the existing "To Do" tab within the Canvas' mobile application, seamlessly integrating its functionality into the online platform. The design not only ensures a cohesive look but also provides a seamless flow across devices used by students. Rooted in a comprehensive design process, my team and I conducted thorough user research, created wireframes, and developed high-fidelity prototypes. Using Figma, we designed user flows for the mobile application, notifications, and online interface.For a more in depth look at the design process and soltions download our proposal.




Website Mockups


Using Figma, we created detailed prototypes for our user flows. We added a "To Do" tab to the dashboard view options, giving users more choices. Inside the tab, there are three filters for customizing the dashboard. We also made sure to include clear checkboxes that match our design of the mobile app.
